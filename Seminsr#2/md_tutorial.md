## Как добавить исходный код


Чтобы добавить исходный код необходимо использовать коснтрукцию следующего вида:
```html
<table>
    <tr>
        <td style = "color: #ff783">
            Текст
        </td>
    </tr>
</table>
```

Пример:

<table>
    <tr>
        <td style="color:#ffd700">
            Текст
        </td>
    </tr>
</table>
